# VFBuilder

Single-page React app for building ships and assembling squadrons for a space dogfighting tabletop game. Data is stored locally in the browser.

## Features
- Ship designer with type-based rules and gun configuration
- Squadron builder with per-ship pilot skill selection
- Local persistence (ships and squadrons)

## Development
1. Install dependencies: npm install
2. Start the dev server: npm run dev

## Notes
- Points tables and upgrade rules are placeholders and can be adjusted when final rules are supplied.
